# AI Chat App

A simple React chat application powered by Google's Gemini AI.

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/ai-chat-app)

## Quick Setup

1. Clone this repository
2. Install dependencies: `npm install`
3. Set your Gemini API key in Vercel environment variables:
   - Go to your Vercel project settings
   - Add environment variable: `REACT_APP_GEMINI_API_KEY` = `your_api_key`
4. Deploy!

## Local Development

1. `npm install`
2. `npm start`
3. Open http://localhost:3000

## Environment Variables

- `REACT_APP_GEMINI_API_KEY`: Your Google Gemini API key
